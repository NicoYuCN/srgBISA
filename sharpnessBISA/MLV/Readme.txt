This package contains Matlab code for implementing the MLV sharpness measure algorithm. 
Please cite the paper followed if you use this code for your research:

K. Bahrami and A. C. Kot, "A Fast Approach for No-Reference Image Sharpness Assessment Based on Maximum Local Variation," IEEE Signal Processing Letter, vol. 21, no. 6, pp. 751–755, 2014.


The main function is MLVSharpnessMeasure.m. This function output the sharpness map and sharpness measure.
